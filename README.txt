README
------

Ferramenta..: https://viromedia.com/viroreact/
Documentação: https://docs.viromedia.com/docs/
Assets......: https://docs.viromedia.com/docs/assets

Dependências: Node, Python2, JDK and The React Native CLI

- Para instalação, habilitação e hello world do Viro React, siga as instruções em:
https://docs.viromedia.com/docs/quick-start-windows
OU
https://docs.viromedia.com/docs/quick-start

- Para executar o exemplo realizado no curso (em anexo):
==> Substitua os arquivos no projeto ViroSample (exemplo que a ferramenta instala)
==> Insira a sua chave de desenvolvedor no arquivo App.js

- Instalar app "Viro Media" em sua loja de aplicativos para testar os códigos via Wifi
===> Para Android, instalar também o app ARCore

- Dispositivos que suportam ARCore (Android 7.0+) ou ARKit (iOS 11.0+)
https://developers.google.com/ar/discover/supported-devices

- Para testar os códigos via USB, siga os passos de "Installing ADB (Android Only)" em:
https://docs.viromedia.com/docs/troubleshooting

- Para gerar Apps ViroReact nativos, siga os passos para configuração das plataformas de desenvolvimento em https://docs.viromedia.com/docs/:
===> Android - PLATFORM SETUP (Android Studio)
===> iOS - PLATFORM SETUP (XCode)
